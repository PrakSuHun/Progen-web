## 1. GPT API란?

GPT API는 OpenAI가 제공하는 언어 모델 API입니다. 코드 한 줄 없이도 다양한 AI 기능을 내 서비스에 붙일 수 있습니다.

```python
import openai

response = openai.chat.completions.create(
  model="gpt-4o",
  messages=[
    {"role": "user", "content": "안녕하세요!"}
  ]
)
print(response.choices[0].message.content)
```

## 2. 프롬프트 엔지니어링

좋은 프롬프트의 3요소:

1. **역할 설정** — "당신은 마케팅 전문가입니다."
2. **맥락 제공** — 배경, 대상, 목표를 명확히
3. **출력 형식 지정** — JSON, 표, 목록 등
