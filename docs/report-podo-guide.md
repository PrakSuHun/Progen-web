# 포도 보고서 작성 가이드

## 용도
**내부적으로 사용할 보고서**입니다.
포도(기존 멤버) 내부에서만 공유되며, 외부에 절대 노출되면 안 됩니다.
포도 관점에서 크루/게스트의 접촉·유입·관계 형성 성과를 분석하는 내부 자료입니다.

## 기본 원칙
포도를 제외한 크루/게스트 데이터 중심으로 분석합니다.
포도는 별도로 "운영진/기존 멤버" 역할로 표기됩니다.

## 금지 단어
- **전도** — 절대 사용 금지 (교회 전도로 오해)
  → "유입", "접촉", "초대", "관계 형성" 사용
- **선교** — 사용 금지
  → "접촉", "유입" 사용
- **생명** (복음 대상 의미) — 사용 금지
  → "크루", "대상자" 사용

## 제목
- "선교 분석 보고서" 금지 → "**포도 분석 보고서**" 사용

## 섹션 규칙
- **기도 제목 섹션 없음** — 완전히 제외
- "전도 전략" → "유입 전략"
- "전도 성과" → "성과 분석"
- "전도 타겟" → "유입 타겟"
- "전도 채널" → "유입 채널"

## 데이터 처리 규칙
- **모든 집계에서 포도 제외**
- 크루/게스트 데이터만 분석
- 포도 참여 현황은 별도 섹션에서 관리용으로만 간단히 언급

## 보고서 구조 (표준)

1. **행사 개요** — 포도 참여 + 크루/게스트 핵심 수치
2. **성과 분석** — 핵심 수치 비교, 누적 크루 현황
3. **대상자 분석** (포도 제외) — 학교별 유입 타겟, 학년별, 성별
4. **유입 경로 분석** — 경로별 신청/출석, 채널 효과
5. **피드백 분석** — 긍정 반응, 가입 관심
6. **포도 참여 현황** — 팀 배정, 포도 역할 점검 항목
7. **다음 행사 전략** — 유입 전략, 행사 개선, 예상 규모
