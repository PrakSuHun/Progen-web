🏠 룸메이트 자료조사룸메이트
나는 대학생이고 '룸메이트 갈등 예방 및 매칭 개선'을 주제로 팀 보고서를 쓰고 있어. 아래를 조사해줘: ① 대학생 룸메이트 갈등의 주요 원인 TOP 3 (수면·청결·생활 패턴 등 — 수치 데이터 포함) ② 국내외 대학에서 실제 도입한 룸메이트 매칭 시스템 사례 2개 (플랫폼명·방식·효과 포함) ③ AI 기반 룸메이트 매칭 서비스 최신 트렌드 (2022년 이후) ④ 이 문제가 해결 안 되면 생기는 실질적 영향 (중도 퇴실률·학업 저하 등 수치 포함) → 텍스트로만 깔끔하게 정리. 팀 토론 후 사이스페이스에 붙여넣을 거야.
📱 갤럭시 자료조사갤럭시
나는 대학생이고 '갤럭시 UX 개선 및 MZ세대 마케팅'을 주제로 팀 보고서를 쓰고 있어. 아래를 조사해줘: ① MZ세대의 스마트폰 선택 기준 TOP 3 (갤럭시 vs 아이폰 비교 데이터 포함) ② 갤럭시 사용자가 가장 불만족하는 기능·UX 요소 (실제 사용자 리뷰·설문 데이터 포함) ③ 삼성이 최근 1~2년 도입한 AI 기반 기능 현황 (갤럭시 AI·서클 투 서치 등, 출처 포함) ④ 경쟁사 대비 갤럭시의 강점과 약점 (점유율·수치 데이터 포함) → 텍스트로만 깔끔하게 정리. 팀 토론 후 사이스페이스에 붙여넣을 거야.

①
입력한 프롬프트
위 프롬프트 그대로 복사해서 보고서에 붙이기
예) "우리 팀 아이디어: 생활 패턴 기반 매칭. 이 아이디어를 뒷받침하는 근거를 찾아줘..."
②
AI 요약 결과
사이스페이스 응답 핵심 2~3줄 발췌 붙이기
예) "미국 Room8 플랫폼은 생활 리듬 기반 매칭 후 갈등 발생률이 43% 감소했다."
③
우리가 반영한 내용 ← 팀이 직접 써야 함
"AI가 알려준 ○○ 사례에서 △△ 기능이 효과적이었다. 따라서 우리는 □□ 방식으로 결정했다." 형태로 작성
예) "Room8 사례에서 수면·청결 데이터 매칭이 효과적이었다. 따라서 우리는 MBTI 대신 생활 리듬 체크리스트 기반 매칭 방식을 채택하기로 결정했다."

프롬프트 1 — 3항 구체화 완성보고서 3항
업로드한 자료를 바탕으로 '3항 아이디어 구체화' 내용을 완성해줘. 우리 팀 아이디어: [아이디어 이름] ① "이 연구에 따르면..." 인용 가능한 문장 1개 ② 해외·국내 유사 사례 연결 2~3줄 ("이미 ○○에서는 이런 방식으로...") ③ 예상 효과 2~3줄 (수치·비교 포함) → 보고서에 그대로 붙여넣을 수 있는 문체로.
프롬프트 2 — 4항 결론·회고 초안보고서 4항
우리 팀 보고서 결론을 써줘. 주제: [룸메이트 / 갤럭시] 최종 아이디어: [이름] 핵심 근거: [가장 인상적인 데이터 1개] 한계·아쉬운 점: [솔직하게 1~2가지] 구성: ① 아이디어 한 문장 정의 ② 필요한 이유 (근거 포함) ③ 실현 가능성·다음 단계 제안 ④ 팀 소감 1~2줄 → 자연스러운 보고서 문체로, 200자 내외.
★ 최종 — 보고서 양식 전체 작성 프롬프트보고서 완성
업로드한 자료를 바탕으로 아래 보고서 양식에 맞게 내용을 채워줘. 팀명: [팀명] 팀원: [팀원 이름] 과제명: [룸메이트 / 갤럭시] == 작성할 항목 == 1. AI 자료조사 요약 → 업로드된 자료에서 핵심 내용 3가지를 각각 2~3줄로 요약. 수치·출처 포함. 2. 해결 아이디어 (표 형식) → 팀 토론 결과를 바탕으로 아래 표를 채워줘: 아이디어 이름 / 방식 / 기대효과 / 별점 (팀 아이디어: [아이디어 이름 / 방식 간단히]) 3. 아이디어 구체화 → 입력한 프롬프트: 위에서 내가 입력한 프롬프트 그대로 → AI 요약 결과: 자료에서 찾은 핵심 사례 2~3줄 → 우리가 반영한 내용: "이 사례에서 배운 점과 우리 아이디어에 어떻게 적용했는지" 2~3줄 4. 결론 및 회고 → 아이디어 한 문장 정의 + 필요한 이유 + 팀 소감 == 출력 형식 == 각 항목 번호와 제목을 붙여서 보고서 문체로 작성. 복붙해서 바로 제출할 수 있게.

이 프롬프트 결과를 복사 → 보고서 파일에 붙여넣기 → "우리가 반영한 내용" 항목만 팀이 직접 수정하면 완성됩니다.
